WARNING:

Normally you create ITFoundedBeepcon (if needed by the client APP) when some report is received via the ITBeepconsManager Class. For example upon receiving a ITBeepconsManagerReportingWithId:Name:Rssi:TxPower:HiId:LowId:MeanRSSI:NumberOfBroadcasts:Mac: During Scanning the SDK creates and maintains it´s own ITFoundedBeepcon objects that can the be retrieved via the GetFoundedPeripherals: method

Properties

AllAdvertiseDataHasReceivedOK

Sometimes not all the advertised information is received OK (name, meanRSSI, TxPower or Identifier). The SDK will report on this field TRUE if all the fields ar OK ore FALSE if some of them are missing.

ErrorDesc

When a connection or other error occurs, this field has a string with a friendlye description of the error

hiid

High Identifier Serial Number, also known as major

identifier

This is NSUUID UNIQUE number that is internally asigned by IOS to this BEEPCON Its only for internal USE because IOS can change randomly this number on time Can be used to index dictionaries etc...

LastDataComputed

When using the internal Engine for media computing, holds a friendly description of last calculus

LastPacketReceivedDate

Date in format NSDate of last packet received time-date (Usefull to determine the time a Beepcon has not been in range This property must be set (if needed) by the Object Creator. On this version it is not included in the constructor method.

loid

Low Identifier Serial Number, also known as minor

LossedPowernum

This is the difference in dB of the meanRSSI and the TxPower. This value is proportional to the distance of the BEEPCON.

meanRSSInum

This is the Mean RSSI (Received Signal Strengt) (dB) measured for this BEEPCON during the Scan Proccess

name

Name of the BEEPCON, as broadcasted

NumberOfBroadcasts

Number of broadcasts recieved of this BEEPCON in the last Scan

NumOfRetries

When the BEEPCON is connected, number of internal retries to make this connection

ProximityDescription

A string descriptor (like Very Close, Close, Far) of the estimated distance to the BEEPCON

TxPowernum

This is the Power Level at 1m that the BEEPCON is transmitting info (dB)

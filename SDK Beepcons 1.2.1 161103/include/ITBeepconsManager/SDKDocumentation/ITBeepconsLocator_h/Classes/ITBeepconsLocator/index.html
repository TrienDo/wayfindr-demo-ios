StartLocatingBeepconsWithHiId:LowId:Identifier:

Call this function to tell IOS to start locating de defined Group of BEEPCONS

- (void) StartLocatingBeepconsWithHiId:(NSNumber *)hiid 
        LowId:(NSNumber *)loid Identifier:(NSString *)identifier;
Parameters
hiid

A 4 byte Number with the also known Major Value. Null to disable filtering by Major

loid

A 4 byte Number with the also known Minor Value. Null to disable filtering by Minor

identifier

a string to identifiy the Group of BEEPCONS being located that will be refered in other functions (Needs to be Unique. Ex : @"com.ilunion.GroupOfBeepconsOneLocated")

Discussion

WARNING:

If loid is supplied (not null), the hiid cannot be null or the Locating Action will be refused

WARNING:

The UUID to locate for will be the last you have set via setUUIDToLocate:

StartReportingBeepconsWithHiId:LowId:RegionIdentifier:

call this function to get the BEEPCONS data reported as soon as IOS encounters the defined Group of BEEPCONS.

- (void) StartReportingBeepconsWithHiId:(NSNumber *)hiid 
        LowId:(NSNumber *)loid RegionIdentifier:(NSString *)identifier;
Parameters
hiid

A 4 byte Number with the also known Major Value. Null to report any Major.

loid

A 4 byte Number with the also known Minor Value. Null to report any Minor

identifier

a string to identifiy the Group of BEEPCONS being reported that will be refered in other functions (Needs to be Unique. Ex : @"com.ilunion.GroupOfBeepconsOneLocated")

Discussion

WARNING:

If loid is supplied (not null), the hiid cannot be null or the Reporting Action will be refused

WARNING:

The UUID to locate for will be the last you have set via setUUIDToLocate:

WARNING:

Apple recommends to use this function in Foreground.

WARNING:

The rate that reports are sended to the client App are controlled internally by IOS.
